快捷键汇总

发表于 2021-05-02 更新于 2024-06-04 分类于 操作技巧

软件操作过程中的快捷键汇总

Windows

快捷键 功能 常用
key1 + key2 + key3
F1 帮助
F2 重命名
Win “开始”屏幕
Win + A 操作中心
Win + B 将焦点放在通知区域
Win + D 显示/隐藏桌面 :heavy_check_mark:
Win + Alt + D 显示/隐藏日期
Win + E 文件资源管理器 :heavy_check_mark:
Win + F 反馈中心
Win + G 游戏栏
Win + H 听写
Win + I 设置 :heavy_check_mark:
Win + K 连接
Win + L 锁定电脑 :heavy_check_mark:
Win + M 最小化所有窗口 :heavy_check_mark:
Win + O 锁定设备方向
Win + P 投屏
Win + R “运行”对话框 :heavy_check_mark:
Win + S 搜索 :heavy_check_mark:
Win + T 循环浏览任务栏上的应用
Win + U 显示设置
Win + V 剪贴板历史记录 :heavy_check_mark:
Win + W 截屏或绘画板 :heavy_check_mark:
Win + X “快速链接”菜单 :heavy_check_mark:
Win + , 显示桌面
Win + . 或 ; 表情符号 :heavy_check_mark:
Win + Pause 系统属性 :heavy_check_mark:
Win + Ctrl + F 搜索电脑
Win + 数字键 切换已经打开的窗口
Win + Shift +数字键 新建数字对应窗口
