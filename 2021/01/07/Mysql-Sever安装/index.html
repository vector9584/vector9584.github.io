    Mysql Sever安装

    发表于 2021-01-07 更新于 2024-06-04 分类于 后端开发

    Mysql的下载以及安装配置

    mysql sever下载官网：https://dev.mysql.com/downloads/mysql/

    选择体量较小的那个zip进行下载

    解压到存放的位置

    打开环境变量

    在path中添加刚刚解压文件夹中的bin目录

    以管理员身份运行cmd

    cd进入sqlsever的bin目录

    cd后的路径不能有空格，如果有，整个路径应该用引号引起来，否则无法进入

    mysqld -install

    缺少msvcp140.dll：链接：https://pan.baidu.com/s/1Xt_nuTTf_-gSQDJgwrpFjQ 提取码：6v5j

    文件存放位置：C/Windows/SYSTEM32下

    应用程序无法正常启动(0xc000007b)：cmd中输入

    显示：Service successfully installed.

    显示：

    2021-01-07T13:26:50.831608Z 0 [System] [MY-013169] [Server] D:.22-winx64.exe (mysqld 8.0.22) initializing of server in progress as process 1528 2021-01-07T13:26:50.862623Z 1 [System] [MY-013576] [InnoDB] InnoDB initialization has started. 2021-01-07T13:26:52.056042Z 1 [System] [MY-013577] [InnoDB] InnoDB initialization has ended. 2021-01-07T13:26:55.123443Z 6 [Note] [MY-010454] [Server] A temporary password is generated for root@localhost: CouYVy/ps5J(

    如果报错Can't create test file……证明cmd没有以管理员方式运行或者命令错误（检查自己的拼写）

    删除data文件夹，重新cmd进入bin，然后按照以下命令

    mysqld -remove MySQL

    mysqld --initialize --console

    mysqld --install

    注意：这里最后一行显示了默认密码

    此时上层目录也多了data文件夹

    net start mysql

    显示：

    MySQL 服务正在启动 . MySQL 服务已经启动成功。

    mysql -u root -p

    显示：Enter password:

    粘贴默认密码并回车

    显示：

    Welcome to the MySQL monitor. Commands end with ; or . Your MySQL connection id is 8 Server version: 8.0.22

    Type 'help;' or ' for help. Type ' to clear the current input statement.

    mysql>

    就成功启动服务并进入了

    接下来修改默认密码

    先修改root用户的密码

    注意后面有分号，最后的字符串就是密码，敲回车后显示

    Query OK, 0 rows affected (0.01 sec)

    表示修改成功

    我们可以退出试一下

    退出命令：exit

    然后敲回车

    显示

    Bye

    然后（按键盘上的↑键可以回到上一个命令）

    mysql -u root -p

    之后输入修改的密码重新进入

    创建自己的用户和密码

    回车后显示：Query OK, 0 rows affected (0.02 sec)

    我们重新退出，用创建的账号登录试一下

    exit

    mysql -u vector -p

    输入密码

    显示：mysql>

    成功进入

    接下来对用户授权

    首先我们要退出当前创建的用户，当前用户权限不够

    然后以root用户进入mysql

    输入命令

    GRANT privileges ON databasename.tablename TO 'username'@'host'

    • privileges：用户的操作权限，如SELECT，INSERT，UPDATE等，如果要授予所有的权限则使用ALL
    • databasename：数据库名
    • tablename：表名，如果要授予该用户对所有数据库和表的相应操作权限则可用表示，如.*
    • host：主机名，用%或localhost替代

    显示Query OK, 0 rows affected (0.01 sec)

    表示该用户拥有了所有权限和所有数据库

    接下来我们就可以用创建的用户进行各种操作了

    在数据库中*.*表示所有数据库中的所有表

    当然如果我们只想把某一个数据库给某一个用户就可以用上面的语法

    如：

    创建数据库：create database test;

    展示数据库：show databases;

    选择某些数据展示：select * from 表名

    当我们有一个数据库叫test，里面有一个表叫data

    show databases;

    展示所有的数据库

    在mysql中用指令use test，就连接上了我们的数据库

    show tables;

    展示所有的表

    select * from data;

    展示表中所有内容

    navicat链接

    连接名：可以用来识别即可

    主机：localhost(自己电脑用localhost，连接其他服务器用其IP)

    端口：3306(默认3306)
