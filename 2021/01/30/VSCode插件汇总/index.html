VSCode插件汇总

发表于 2021-01-30 更新于 2024-06-04

vscode可高度自定义，此部分内容不断更新

插件名称 功能
Auto Rename Tag 标签对前后自动同步修改
Atom One Dark Theme 舒适的主题
Bracket Pair Colorizer 彩虹括号
Chinese (Simplified) Language Pack for Visual Studio Code 中文环境
CodeSnap 代码截图
filesize 左下角显示当前文件大小
HTML CSS Support 智能提示CSS类名以及id
HTML Snippets html语法补全、高亮
Live Server 网页端口调试，可实时修改显示（前提是文件处在文件夹中，并设置文件自动保存）
open in browser 可以方便的在浏览器中打开html
Power Mode 输入特效
Rainbow CSV csv语法高亮
TabOut tab键跳出符号对
vscode-icons vscode文件图标主题
YAML yml文件语法高亮
CSS Peek 从html直接跳转css进行编辑
ESLint 插件式架构，有多种主流的编码风格规则集可供选择，典型的有 Airbnb、Google 等，你甚至可以攒个自己的，按下不表，它的规则在.eslintrc.json里配置；
Beatufy 一个jsBeautifier的插件，支持JavaScript、JSON、CSS和HTML。可通过.jsbeautifyrc文件自定义。它是最流行的格式化工具，目前有230万的下载量。
会了吧 打开源码可以自动分析所有包含的英语单词，并显示解释结果
Code Spell Checker 对大部分非英语母语又不想写出四不像变量名的程序员来说，正确识记拼写各种单词还是有不小的挑战，比模棱两可时需要去查在线词典不同的是，这款插件能实时的识别单词拼写是否有误，并给出提示，不少 bug 都是因为拼写错误导致的
formate: CSS/LESS/SCSS formatter css文本格式化
Beautify 格式化文本
Easy LESS 将less文件自动生成css文件，方便引入
korofileheader 生成函数注释

快速复制某一行：shift+alt+↓

交换某两行：alt+↑或alt+↓

选择多个相同的单词：ctrl+d

添加多个光标：ctrl+alt+↑或ctrl+alt+↓///或者按住鼠标滚轮滑动

选择某个区块：shift+alt然后拖动鼠标（起始光标位置很重要）

全局替换：ctrl+h

快速定位到某一行：ctrl+g

放大缩小编辑器界面：ctrl++或ctrl+-

单行注释：ctrl+/

多行注释：shift+alt+a（改为ctrl+shift+/）

css模糊搜索class

[class^="部分类名"]{

background……

}

批量添加缩进：选中按tab

批量删除缩进：选中按shift+tab

变量名上右键（或f2），重命名符号，可以统一修改，shift+enter可以预览

字段上右键，更改所有匹配项（ctrl+f2），统一修改相同字段

右键，源码操作，可以对导入进行排序、整理import语句，删除所有未使用的代码，添加所有缺少的导入等

格式化：alt+shift+f

注释插件：KoroFileHeader

添加文件注释：ctrl+win+i

添加函数注释：ctrl+win+t

图案注释：f1 codestyle

注释官方说法：JSDoc

括号对指南就是参考线的彩虹色
