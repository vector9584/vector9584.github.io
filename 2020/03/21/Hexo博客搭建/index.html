Hexo博客搭建

发表于 2020-03-21 更新于 2024-06-04

从零开始搭建一个属于自己的博客，内容随着学习会不断更新，很多术语仅根据自己的理解进行解释，如有偏差望自行考证。

环境：windows

处理器：64

hexo博客比较成熟，网上有很多问题解决觉得办法，因此选用hexo框架进行博客搭建

部署环境配置

Nodejs

Nodejs是管理js包的一个软件，你可以通过nodejs在网上下载js包并进行管理。

安装过程如果没有特别的需求，默认下一步即可。

安装完成后，电脑将会有两个文件：Nodejs本身以及npm包管理器。

检查是否安装成功

都显示版本号即为安装成功

本身我们通过npm安装，但是国内镜像源很慢，所以我们利用npm来安装cnpm（淘宝镜像源）

等待自动安装即可

敲回车后会自动补全即为安装成功

也可查看版本号

建立一个空的文件夹，用来存储自己的博客文件

Git

git是代码管理器，用以控制版本号，搭建hexo的代码要传到远端服务器需要用git来控制并操作

git下载并安装

安装过程中如果没有特殊需求，默认下一步即可

我们在之前建立的用以存放hexo博客文件的文件夹上鼠标右键，选择：Git Bash Here

此时会出现命令行窗口，并且路径为当前文件夹的绝对路径

远端部署

接下来我们要把博客部署到远端

有两个选择：github、码云

这两个都是代码管理终端，新注册的账号提供一个pages服务：把你的代码以网页形式渲染展示，免费的一般都仅有一个仓库提供这个服务。

此处我比较推荐码云，首先服务器在国内，访问比较快，其次是中文界面，比较友好。
