Markdown

发表于 2020-03-21 更新于 2024-06-04

前言

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。Markdown的好处在于 - 易读、易写、易改动 - 兼容HTML - 跨平台使用，并且越来越多的网站支持Markdown

Markdown具有一系列衍生版本，用于扩展Markdown的功能（如表格、脚注、内嵌HTML等等），这些功能原初的Markdown尚不具备，它们能让Markdown转换成更多的格式，例如\(\LaTeX\),Docbook。Markdown增强版中比较有名的有Markdown Extra、MultiMarkdown、Maruku等。这些衍生版本要么基于工具，如Pandoc；要么基于网站，如GitHub和Wikipedia，在语法上基本兼容，但在一些语法和渲染效果上有改动。（百度百科）

书写工具

Typora

Typora是一个所见即所得的编辑工具，界面简洁，使用方便。可进行错别字标注。可以方便的插入图片与表格，语法自动补全，更有多个主题可供选择。写文章时我比较喜欢\(Academic\)主题，在光线较暗时我喜欢用\(Night\)主题。

Typora支持Pandoc插件，可以支持更多格式的导出，若要支持内联公式需要在偏好设置中勾选。

常用功能 快捷键
加粗 Ctrl+B
倾斜 Ctrl+I
下划线 Ctrl+U
超链接 Ctrl+K
插入表格 Ctrl+T
删除线 Alt+Shift+5
着重关键字 Ctrl+Shift+`
清除格式 Ctrl+\
选中一行 Ctrl+L
侧边栏 Ctrl+Shift+L

Visual Studio Code

Visual Studio Code是一款轻量级的编辑器，本身支持Markdown，只需要新建一个文件类型为md的文件即可书写和预览。但是为了更方便的使用经常会搭配一些插件使用。

插件名称 主要功能
Markdown Preview Enhanced 预览、导出、插入图片
vscode-pdf 支持PDF查看
markdown-all-in-one 快捷键、目录、预览
Markdown TOC 生成目录
Markdown Math 支持内联公式
Markdown PDF PDF输出
EverMonkey 上传印象笔记
Markdown Theme Kit 主题

本人主要将其作为查看器，其功能性可以自行探索，此处不多赘述。

Markdown一般情况下支持HTML语言，HTML几乎能完成Markdown的格式操作，在此处重复功能就不过多赘述，通过学习HTML和CSS可以对Markdown的排版进一步优化，上一部分内容中为常用的基本语法，其中包括了一些HTML语言，此部分内容会根据自身的HTML与CSS学习情况不断更新。
